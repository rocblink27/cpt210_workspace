## Overview
Welcome to the Information & Computer Technologies Department's *CPT-210: Operating Systems and Peripherals* course.

This workspace is designed to support development work using Visual Studio Code (VS Code) and the Raspberry Pi. 

We’ll be using VS Code's **Remote - SSH** extension to connect to the Raspberry Pi in a **headless configuration**.

This workspace is pre-configured to support that workflow and includes the tools and resources you’ll need for your assignments and projects throughout the semester.
